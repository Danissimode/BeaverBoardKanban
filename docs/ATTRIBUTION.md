# Attribution

Beaver Board is an independent product forked from the KittyClaw project.

## Origin

Beaver Board was originally derived from **KittyClaw** by the same author. The original project served as a local-first Kanban board with AI agent orchestration. Over time, Beaver Board evolved into a separate product with its own identity, packaging, and public distribution.

## What Changed

- **Product identity**: Renamed from KittyClaw to Beaver Board for public distribution.
- **Packaging**: Beaver Board is distributed as a standalone macOS `.app` and DMG, not a repository clone.
- **Storage paths**: Migrated from development-specific paths to standard platform directories (`~/Library/Application Support/BeaverBoard`).
- **Upstream policy**: KittyClaw remains an upstream source for selected improvements (security fixes, bug fixes, board features), but Beaver Board does not automatically pull all upstream changes. Each upstream commit is reviewed, classified, and adapted before adoption.
